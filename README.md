# MotoTourWally
Project in progress ... 


MERN project for Moto tours "Wally" , a company that offers tours in motorcycles around the world. 

Use of: 
- React for Client side 
- Express / Node.js for server side 
- MongoDB (Atlas) for DB
- User authentication (bcrypt and JWT)

To start project:
- npm i 
- npm run dev
